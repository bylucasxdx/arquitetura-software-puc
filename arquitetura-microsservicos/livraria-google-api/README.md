## Como iniciar o projeto

```
# Clone o repositório
git clone https://github.com/bylucasxdx/livraria-google-api.git

# Entre na pasta
cd livraria-google-api

# Instale as dependências
pip install -r requirements.txt

# Start da aplicação 
python main.py
```
